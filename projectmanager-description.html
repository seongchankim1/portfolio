Project Manager 프로젝트 소개

한 줄 소개

📌국내 사용자를 고려한 친숙한 UI의 칸반보드 협업 플랫폼
🕒2024.12 ~ 2025.04
👥4명 (백엔드 2, 프론트엔드 2)
🔗 🔗 GitHub 📄 Page

Trello나 Jira와 같은 칸반보드 툴은 강력하지만, 국내 사용자에게는 인터페이스가 낯설고 접근성이 떨어지는 경우가 많습니다. 우리는 “누구나 쉽게 사용할 수 있는 협업 툴”을 만들자는 목표로 Project Manager를 기획했습니다.

이 프로젝트는 소규모 팀이나 사이드 프로젝트 환경에서 보다 직관적이고 빠르게 협업할 수 있도록 설계되었으며, 실시간 초대, 할 일 관리, 알림 수신함 등 실무 친화적인 기능을 중심으로 구현되었습니다.

다양한 프로젝트와 멤버들을 한 화면에서 관리하고, 사용자마다 개인 맞춤형 진행 상황을 확인할 수 있어 단순한 칸반보드를 넘어 협업 효율을 극대화하는 툴로 발전시켰습니다.

주요 기능

소셜 로그인
  • 카카오 로그인으로 누구나 쉽게 접근
  • JWT 기반 인증 처리로 보안성과 편의성 확보
프로젝트 초대
  • 초대 코드 및 알림을 통해 팀원을 프로젝트에 초대
  • 초대 수락 시 자동 권한 할당 및 참여
알림 수신함
  • 초대, 권한 변경, 카드 이동 등 주요 이벤트 실시간 알림
  • 읽음 처리 및 알림 기록 기능 포함
개인별 진행도 관리
  • 개인에게 할당된 카드만 필터링하여 확인 가능
  • To-Do / In-Progress / Done 칼럼 이동으로 직관적인 상태 관리
