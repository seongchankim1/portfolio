서비스 환경

  • Hosting: AWS Lightsail
  • CMS: WordPress (PHP 기반)
  • DB: MySQL
  • Cache: Redis
  • Frontend: WordPress + Custom Theme

기술 선택 이유 및 구현 방식

WordPress + PHP
  • 기존 CMS 기반에서 빠른 MVP 개발을 가능하게 함
  • 정적 페이지(Page)와 동적 콘텐츠(Post)를 분리하여 유연한 게시 시스템 구축
  • 회원 기능, 카테고리, 태그 등 커뮤니티 기능을 손쉽게 제공
MySQL
  • WordPress 기본 스키마가 MySQL에 최적화되어 있어 추가 설정 없이 운용 가능
  • 카테고리, 태그, 사용자 정의 필드(meta) 등 다양한 연관관계 데이터를 안정적으로 처리
Redis – 실시간 인기 게시물 집계
  • 조회수, 좋아요 등 실시간 통계를 빠르게 계산하기 위해 Redis Sorted Set을 활용
  • 1주일 단위로 자동 만료(Expiration)되도록 설계하여 캐시 유지 부담 최소화
백오피스 – 통계 대시보드
  • MySQL + Redis 조합으로 실시간/누적 데이터를 분석하여 관리자 페이지에 시각화
  • 브랜드, 셀럽, 상품 분류별 Top N 추출 → CSV/스프레드시트 변환 기능 제공
다국어 지원
  • WordPress의 다국어 플러그인(Polylang)을 통해 한국어/영어 페이지 분기 처리
  • 향후 글로벌 확장을 고려해 콘텐츠/경로/링크 등을 언어별로 구분 설계
