📌실시간 경매를 통한 중고거래 플랫폼
🕒2024.07 ~ 2024.08
👥5명 (백엔드 전담)
🔗 🔗 GitHub 📄 Page

팀 프로젝트를 진행하며, 우리는 일상에서 쉽게 지나치는 중고 거래 방식에 주목했습니다. 요즘은 불필요한 물건을 중고마켓에 올리는 일이 많아졌지만, 가격 책정이 어렵거나 거래 타이밍을 놓쳐버리는 경우가 많았습니다. 이 문제를 해결하고자, 실시간 경매 기반의 중고 거래 플랫폼 ‘itsmine’을 기획하게 되었습니다.

사용자는 불필요한 물건을 경매에 등록하고, 실시간으로 입찰을 받아 가장 높은 가격에 자동 판매할 수 있습니다. 반대로, 구매자는 경쟁 입찰을 통해 적정 가격에 상품을 획득하거나, 즉시 구매가로 바로 거래할 수도 있습니다.

실시간으로 진행되는 경매와 채팅 시스템은 사용자 간 신뢰를 높이며, 마감 시간과 입찰 현황을 손쉽게 확인할 수 있어 기존 중고마켓에서 경험할 수 없는 짜릿한 거래 경험을 제공합니다.

주요 기능

실시간 경매
  • 사용자는 원하는 상품을 등록하고, 제한 시간 내 최고가 입찰자에게 자동 판매됩니다.
  • 입찰이 들어올 때마다 실시간으로 남은 시간을 갱신하여 거래 몰입감을 높입니다.
즉시 구매
  • 구매자는 즉시 구매가로 다른 경쟁자보다 먼저 거래를 완료할 수 있습니다.
  • 경매 마감까지 기다릴 필요 없이 빠르게 물건을 확보할 수 있어 편리함을 제공합니다.
실시간 채팅
  • 판매자와 구매자는 상품에 대한 문의를 실시간으로 주고받으며 신뢰도 높은 거래를 유도합니다.
  • 경매 중에도 자유롭게 소통하며 거래 조건을 조율할 수 있습니다.

사용자 관점

  • “버리기엔 아깝지만, 사용하진 않는 물건이 있다면?”
    → 경매에 등록하고, 입찰을 기다려보세요. 기대 이상의 가격에 판매될 수 있습니다.
  • “싸고 괜찮은 중고물건을 찾고 있다면?”
    → 경매에 올라온 다양한 물건들을 확인하고, 실시간으로 입찰해보세요. 누가 먼저 가져갈지 모릅니다!
